## 🛠 usage

### setting up back-end
```sh
$ git clone https://github.com/alok722/contact-management.git

$ cd backend

$ npm i

$ npm run start
```

### setting up front-end
```sh
$ git clone https://github.com/alok722/contact-management.git

$ cd frontend

$ npm i

$ npm run start
```
